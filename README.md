# Single Page Markdown

这是一个**纯静态**单页应用，通过 [markdown-it](https://github.com/markdown-it/markdown-it) 将 Markdown 字符串渲染为可视化网页内容。

你只需要修改 `index.html` 中的 `markdownContent` 变量，填入你想要展示的 Markdown 内容，然后直接在浏览器中打开这个 `index.html` 页面，就可以看到渲染后的效果了。

## 为什么需要这个？

因为我经常有想要把 Markdown 内容快速展示出来的需求，但又不想搭建一个完整的 Markdown 编辑器或者博客系统。这个单页应用提供了一个简单、快速的解决方案，让你可以轻松地将 Markdown 内容转换为网页格式，适合用于个人笔记、文档展示或者任何需要快速渲染 Markdown 的场景。
